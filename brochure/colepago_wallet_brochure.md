# ColePago Wallet – AI Features & Technology Stack

---

## 🚀 AI & Smart Features

- **Face Detection (Google ML Kit):**
  - Uses the selfie camera to detect faces.
  - Ensures only one face is present for secure transactions.
  - Obscures actions if multiple faces are detected.
- **Biometric Authentication:**
  - Supports fingerprint, face, and iris recognition.
  - PIN fallback if biometrics unavailable.
- **Notifications:**
  - Real-time transaction and activity alerts for parents and kids.

## 💳 Payment & Security

- **🛡️ NO REAL MONEY EXPOSURE IN BACKEND (CRUCIAL SECURITY):**
  - **ONLY VIRTUAL TOKENS/COINS FLOW THROUGH THE BACKEND.**
  - All real money transactions (card payments, wallet funding) are securely handled by Stripe & Mercado Pago.
  - The backend never stores or processes real money or card data—just virtual wallet balances and token transactions.
- **Stripe & Mercado Pago Integration:**
  - Secure, modern payment SDKs for wallet funding and transactions.
- **Parent/Child Authentication:**
  - Role-based access and secure login for both parents and children.

## 🛠️ Backend Stack

- **FastAPI:** High-performance Python backend for APIs.
- **PostgreSQL:** Reliable relational database for wallet and transaction data.
- **Docker:** Containerized deployment for easy scaling and portability.
- **Terraform:** Infrastructure as code for cloud provisioning and management.

## 📱 Mobile App Stack

- **Flutter:** Cross-platform mobile app framework (Android/iOS).
- **Dart:** Modern language powering the Flutter app.
- **Camera & ML Kit:** For face detection and camera-based features.
- **Local Auth:** For biometric authentication.
- **Shared Preferences:** For secure local storage.
- **Dio & HTTP:** For robust API communication.
