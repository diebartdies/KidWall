# ColePago Backend (KidWall)

A FastAPI backend for the ColePago (KidWall) kids' wallet system.

## Features
- Parent/child authentication
- Wallet with spending buckets
- Parent funding (with 2% public commission)
- Language selection (default by country)
- Transaction history
- Notifications
- Future payment API integration
- PostgreSQL, Docker, Terraform
- Colorful backpack logo branding

## Quick Start
1. Copy `.env.example` to `.env` and set your secrets.
2. `docker-compose up --build`
3. Access API docs at `http://localhost:8000/docs`
